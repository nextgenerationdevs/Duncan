          Функция apply принимает два аргумента, первым должен быть объект, который послужит контекстом (this) для функции у которой вызван apply, а второй аргумент должен быть массивом, чьи элементы послужат аргументами функции у которой вызван apply.

          В твоём случае ты вторым аргументом ставишь не массив, а функцию. По идее тебе достаточно просто обернуть function(a, b) { return a + b; } в квадратные скобки, чтобы получился массив с одним элементом, которым и будет функция, но в данном случае вместо apply лучше подойдёт call, так как кол-во аргументов фиксировано.

                                                                                                                                                      первая функция "sum" возвращает сумму элементов псевдомассива arguments, перебор этих элементов она осуществляет через метод "reduce", но так как у псевдомассива arguments нет этого метода - мы "занимаем" его у обычного массива через явное указание контекста методом "call". Аналогично работает и функция "mul". Функция "applyAll" имеет один параметр (func), в этот параметр в качестве аргумента передается имя функции(первый аргумент при вызове), которой мы будем передавать остальные аргументы. Эти аргументы мы получаем из псевдомассива arguments, поэтому в строке с третьей меткой " * " мы возвращаем значение функции имя которой содержится в параметре "func", передавая ей аргументы в виде массива. [].slice.call(arguments, 1) здесь занимаем у массива метод "slice" и вызываем его в контексте псевдомассива arguments, копируя элементы arguments в новый массив, начиная со второго элемента (второй аргумент в "call" = 1, а не 2, потому что индексация эл. начинается с 0)
